# Sell Products

# Showcasing use of 
* Ruby on Rails
* Custom User Login/Registration
* Authentication/Validation for Privileged Actions
* Advanced Active Record Queries

From wireframe to deployed in under 3.5 hours. This website allows user to register or login (with validations performed through the model) and create groups which other users can join. A user can perform CRUD operations with the groups and the page is made using restful routes.
