# Sell Products

# Showcasing use of 
* Ruby on Rails
* Custom User Login/Registration
* Authentication/Validation for Privileged Actions
* Advanced Active Record Queries

Website where users can add products and list a price. Other users can view the listings and buy these products (simulated). The transaction table tracks all transactions. User dashboard shows all products and transactions of the user in three categories avaialble, sold by the user and purchased by the user.
